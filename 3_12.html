Course Summary: Database Management System (DBMS)

Unit 1: Introduction to DBMS

Overview of databases, advantages of DBMS over file system, data models, schema & instances, DBMS architecture, data independence.

Unit 2: Entity Relationship Model

Entity types, attributes, relationships, ER diagrams, mapping ER model to relational model, constraints.

Unit 3: Relational Model & SQL

Relational algebra, relational calculus, SQL basics (DDL, DML, DCL), constraints, joins, subqueries.

Unit 4: Normalization

Functional dependencies, anomalies, normalization (1NF, 2NF, 3NF, BCNF), decomposition.

Unit 5: Transactions & Recovery

Transaction concepts, ACID properties, concurrency control, deadlock handling, recovery techniques.
